# Instructions

Make sure to complete Fastfetch instructions first.

Go to the profile folder for powershell. To find it, type this command:
```powershell
$PROFILE
```
If the location is empty, type the following command:
```powershell
New-Item -Path $profile.CurrentUserAllHosts -Type File -Force
``` 

Make sure to set your username for the path for `config.jsonc` from Fastfetch
