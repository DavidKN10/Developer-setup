# Instructions

Make sure to complete Terminal instructions first.

Install through winget:
```powershell
winget install fastfetch
```
Create a folder `.config` in `C:\Users\{Username}\`

Inside `.config`, create a folder named `fastfetch`

Place `ascii.txt` and `config.jsonc` inside the `fastfetch` folder.

In `config.jsonc`, make sure to set your username in `source`.
